# 📐 PPT 设计建议

## 整体风格
- **主色调**：蓝色（科技感）+ 橙色（活力、旅游）
- **字体**：标题用粗体（如思源黑体 Bold），正文用常规字重
- **图标**：使用扁平化图标，推荐 Flaticon、iconfont

## 页面布局建议
- 每页内容不超过6个要点
- 数据用大字突出展示
- 对比用表格或左右分栏
- 流程用时间轴或箭头

## 图表建议
- 市场规模：柱状图
- TAM/SAM/SOM：同心圆或漏斗图
- 收入来源：饼图
- 增长趋势：折线图
- 用户画像：人物卡片

## 配图建议
- 封面：城市天际线/自然风光
- 产品页：APP截图/手机Mockup
- 用户页：多元化人群照片
- 团队页：真实团队照片
